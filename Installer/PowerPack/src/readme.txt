FreeArc PowerPack contains executables required for use of external
compression methods in FreeArc. It should be installed in order to compress
archives with -max switch and extract back such archives.

After installation, directory "C:\Program Files (x86)\FreeArc\PowerPack\max"
will contain a few executables required for -max option support, while directory
"C:\Program Files (x86)\FreeArc\PowerPack\bin" will include much more
executables that may be used by custom compression settings.
